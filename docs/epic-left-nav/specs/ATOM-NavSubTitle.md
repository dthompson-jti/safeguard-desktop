# Component Spec: Left Navigation Sub-title

## 1. Overview
A non-interactive heading element used to categorize items within a section.

## 2. Terminology & HTML
*   **Figma Name**: `Left Navigation Sub-title`
*   **Suggested Tag**: `<h3>` or `<h4>`
*   **ARIA**: `role="heading"` (implicit).

## 3. Geometry & Dimensions
*   **Height**: 28px (approx, based on line-height).
*   **Padding**: 
    *   Horizontal: 34px (Indented from section edge).
    *   Vertical: 4px.

## 4. Visual Tokens
| Property | Value | Token Reference |
| :--- | :--- | :--- |
| **Text Color** | `#436289` | `var(--control-fg-faint)` |
| **Font Style** | Regular (400) | `Inter` |
| **Font Size** | 14px | `var(--font-size-sm)` |

## 5. Keyboard Navigation
*   **None**: This element is not focusable or interactive.
