# Component Spec: Divider

## 1. Overview
A thin horizontal line used to separate groups.

## 2. Geometry & Dimensions
*   **Height**: 1px.
*   **Width**: Full container width (or with standard margins).

## 3. Visual Tokens
| Property | Value | Token Reference |
| :--- | :--- | :--- |
| **Color** | `#D9DFE5` | `var(--surface-border-faint)` |

## 4. Implementation
*   Use `<hr />` or a `div` with a border-top.
